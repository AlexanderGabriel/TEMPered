Calibration tool

For this tool, input the actual temperature of the reference material (e.g. 100 for boiling water) in the "Actual temperature" column, and input what the sensor said at that temperature in the "Measured temperature" column on the same line.

You only need to input data for as many measurements as you have; the remaining lines can be left empty.

Below the inputs, you'll be given the parameters to give to tempered to have it calibrate this sensor's measurements and print the calculated actual temperatures.

This tool uses the polynomial interpolation method, as described on the example derivation page in the wiki, with all its limitations. However, the polynomials the tool supports have a low enough order that it should work well enough for most cases.
